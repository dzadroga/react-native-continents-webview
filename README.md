# Continents WebView Application

## Overview
This project was developed as part of my Mobile Application Development coursework using React Native. The application allows users to explore information about the seven continents by selecting a continent and viewing its corresponding Wikipedia page within an embedded WebView.

## Features
- Interactive list of all seven continents
- Navigation using buttons for each continent
- Embedded WebView to display external websites
- Dynamic state management for continent links
- Seamless in-app browsing experience

## Technologies Used
- React Native
- JavaScript (ES6+)
- WebView API
- Android Emulator / iOS Simulator

## What I Learned
- Implementing WebView components in React Native
- Managing application state with arrays of objects
- Handling navigation between views
- Integrating external web content into mobile apps

## How to Run
1. Clone the repository
2. Navigate to the project folder
3. Install dependencies:
   npm install
4. Run the application:
   npm start
6. Launch the emulator
   npm run android
